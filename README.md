# algorand-rust

Demo implementation of Algorand Agreement (2018)
https://eprint.iacr.org/2018/377.pdf

Demo:
```
git clone https://github.com/peitalin/algorand-rust;
cd algorand-rust;
cargo run;
```

Directory Structure:
```
"src"
├── "main.rs"
└── "votes"
    ├── "message_type.rs"
    ├── "mod.rs"
    ├── "signatures.rs"
    └── "vote.rs"
```
